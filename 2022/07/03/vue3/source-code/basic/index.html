知识储备

阅读后文需要以下知识储备：

  • ESNEXT 最新规范

  • 理解 vue2 文档中编写的响应式原理

  • pnpm 和其 workspace 的使用，了解 pnpm link 指令

本人开发环境

  • 操作系统：Windows11 x64 专业版

  • IDE：vscode 最新稳定版

  • 调试项目 package.json 依赖版本

版本要求

  • node：>= v16.5.0

  • vue 源码：v3.2.37

  • 包管理工具 pnpm：>= 7.5.0
